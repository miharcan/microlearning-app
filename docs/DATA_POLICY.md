# Data Distribution Policy

## Current State
- Full production content banks are private.
- Raw/intermediate generation artifacts are private.
- Public repo contains only showcase-safe artifacts.

## If We Publish MCQ Packs Later
Before any public dataset release:
1. Validate source licensing and attribution requirements.
2. Separate original vs. derived content clearly.
3. Publish a dedicated dataset license and attribution file.
4. Remove quality/policy internals that expose competitive IP.

## Commercial App Note
A paid app can use private content and private pipelines.
Publicly releasing datasets is optional and can be done in a separate controlled package.
